Spider
 Install
Last updated: 10/5/2014

The Contensive spider is a background process that indexes the content on your site. As it indexes, it also finds broken links and html problems. It is automatically included when you install other Add-ons that use the data, such as the Text Search 2. To modify its settings, use the Spider Settings link installed on the navigator under Manager Add-ons >> Spider. Install Adobe Acrobat on the server and it will index pdf documents.

1/21/2011 - Added support for duplicate blocking (see text search 2).

4/22/2011 - Added Site Warning reporting.

Text Search 2
 Install
Last updated: 5/7/2012

Use Text Search to add a full text search feature to your website. When you install it on your site, it uses the Spider Add-on to begin fetching and indexing all the pages on your site, one per minute. They are updated daily, and right after they are edited. To use it, select or add a page you want to use as your search results page and drop this add-on on the page. You can also add it to other pages, or your template, and use its options to select which page will display results. There are others options to include images and result text in the search results. Add any custom text search form by simply submitting the form input 'textsearchwordlist' to the page with this add-on.

1/21/2011 - Added optional Topic and Section advanced search options. Added blocked sections so the search can be limited to specific site sections. Added duplicate blocking if page content and page title are the same.
